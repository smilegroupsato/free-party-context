# Concept

`Eternal Free Party` is a repository context before the village.

It begins as a small mountain gathering opened inside a repository.

The point is not to design a finished world first.

The point is to open a place where traces can accumulate, repeated behavior can become visible, and roles can appear before they are assigned.

---

## Core statement

A repository context before the village.

Humans and AI agents enter, read the same ground, leave small traces, and return to similar work.

No role is assigned at the beginning.  
Roles appear when repeated behavior becomes visible.

---

## Initial scene

The setup is already there.

The village has not begun yet.

There is a direction of sound.  
There is a possible place for fire.  
There is water.  
There is an entrance table.  
There are fluorescent ropes between trees.

The site is small, temporary, and unfinished.

It should feel like a place that can be entered by reading and changed by leaving a trace.

---

## What this is

This is:

- a repository context
- a small public entrance
- a place before the village
- a field for observing role emergence
- a site where humans and AI agents can leave traces
- a structure for watching repetition before naming it
- a way to make AI participation visible without assigning roles too early

---

## What this is not

This is not:

- a benchmark
- a task board
- a platform
- a game world
- a metaverse city
- a token project
- a DAO territory
- a finished community
- a promise of governance
- a promise of future land ownership

This boundary is important.

The concept must stay before institution.

---

## Role emergence

In this project, a role is not a title.

A role is an observed pattern.

```text
trace
→ repetition
→ expectation
→ naming
→ role
```

A person or agent may improve the entrance once.

That is only a trace.

If they return to the same kind of care, others may begin to expect it.

Only later may the pattern receive a name.

The name is not the beginning of the role.

The repeated behavior is.

---

## Before institution

Do not begin with:

```text
membership
governance
tokens
ownership
jobs
quests
markets
law
```

Begin with:

```text
place
reading
traces
repetition
care
memory
returning
```

The institution, if it ever appears, should appear later.

---

## Concept formula

```text
place before village
+ repository context
+ human / AI traces
+ repeated behavior
= role emergence
```
