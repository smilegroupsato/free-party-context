# Flyer History

The live files in `flyer/` are the current public flyer.

The directories below this one preserve earlier accepted states. They are not discarded drafts. They are part of the history through which the flyer became visible.

## Versions

### v0.0 — text before image

The first public flyer used only typography, color, and structure.

It established the first entrance:

- Flyer
- Venue
- Role

### v0.1 — Phase 0: the path and the lights

The first image entered the flyer.

The scene contains:

- no people
- a mountain path
- a few temporary lights
- a place still becoming visible

Its public rhythm is:

- Observe
- Record
- Return

## Versioning rules

1. The files in `flyer/` represent the current accepted state.
2. An accepted historical snapshot is not overwritten by a later design.
3. A new version is created when the visible story, atmosphere, or structure changes meaningfully.
4. Small corrections, accessibility fixes, and deployment repairs do not require a new version by themselves.
5. Hero images and their HTML/CSS state use the same version number.
6. Image assets use versioned filenames under `flyer/assets/`.
7. The version sequence records emergence; it is not a release train.

The flyer may change because the party changed, because a trace became visible, or because repeated attention revealed a new shape.

# フライヤー履歴

`flyer/` 直下のファイルが、現在公開されているフライヤーである。

このディレクトリ以下には、以前に受け入れられた状態を保存する。没案置き場ではない。フライヤーが見えるようになるまでの履歴そのものである。

## バージョン

### v0.0 — 画像より前の言葉

最初の公開フライヤーは、文字、色、構造だけで作られた。

最初の入口として、次の三つを置いた。

- Flyer
- Venue
- Role

### v0.1 — Phase 0：道と灯り

最初の画像がフライヤーへ入った。

景色にあるものは、次の通りである。

- 人はまだいない
- 山の道がある
- 仮設の灯りがいくつかある
- 場所はまだ完全には見えていない

公開面のリズムは、次の三つになった。

- Observe
- Record
- Return

## バージョン運用

1. `flyer/` 直下を、現在受け入れられている公開状態とする。
2. 履歴として確定したスナップショットは、後のデザインで上書きしない。
3. 見えている物語、空気、構造が意味のある形で変化したとき、新しいバージョンを作る。
4. 軽微な修正、アクセシビリティ改善、配信上の修復だけでは、原則として新バージョンにしない。
5. Hero画像と、そのHTML/CSS状態には同じバージョン番号を使う。
6. 画像アセットは `flyer/assets/` にバージョン付きファイル名で置く。
7. バージョン列は発生の記録であり、定期リリース工程ではない。

Partyが変わったとき、痕跡が見えるようになったとき、あるいは反復する注意が新しい形を発見したとき、フライヤーも変わりうる。
